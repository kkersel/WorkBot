# WorkBot — Telegram-бот для графика работы

Бот для отслеживания графика работы в чате. Показывает кто работает, кто отдыхает, и когда у всех общий выходной.

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и справка |
| `/set 3/2` | Установить график (3 рабочих/2 выходных) |
| `/status` | Кто сегодня работает/отдыхает + общий выходной |
| `/dayoff` | Ближайший общий выходной |
| `/unemployed` | Статус безработного 😎 |
| `/work` | Вернуться к графику |
| `/sync` | Синхронизировать участников чата |

## Установка

### 1. Создать бота в Telegram

1. Открой [@BotFather](https://t.me/BotFather)
2. Отправь `/newbot`
3. Придумай имя и username (должен заканчиваться на `bot`)
4. Скопируй токен

### 2. Настроить проект

```bash
# Установить зависимости
pip install -r requirements.txt

# Создать файл с токеном
cp .env.example .env
```

### 3. Вставить токен

Открой `.env` и вставь токен:
```
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

### 4. Запустить

```bash
python bot.py
```

## Использование в чате

1. **Добавь бота в чат**
2. **Сделай бота админом** — для получения списка участников
3. **Отправь `/sync`** — бот добавит всех участников как безработных 😎
4. **Кто хочет график** — отправляет `/set 3/2`
5. **`/status`** — покажет кто работает, кто отдыхает, и общий выходной

## Примеры использования

```
/set 3/2 25.03.2026             # Старт с 25 марта 2026
/set 3/2 01.03                  # Старт с 1 марта (текущий год)
/set 3/2                        # Старт с сегодня
/set 4/2 2026-03-15             # Формат YYYY-MM-DD
/status                         # Кто сегодня работает + общий выходной
/dayoff                         # Когда у всех выходной
/unemployed                     # Я безработный 😎
/work                           # Снова работаю
/sync                           # Обновить список участников чата
```
